# 📦 File Compression & Decompression Project

Welcome to the **File Compression & Decompression Project**! This project provides a simple way to compress and decompress files using Python. It's perfect for learning how file compression works and for managing file sizes efficiently.

---

## 📁 Project Overview

This project allows you to:

- **Compress** files to reduce their size.
- **Decompress** files to restore them to their original form.

By compressing files, you can save storage space and make file transfers faster. Decompression lets you access the original content whenever needed.

---

## 🛠️ Features

- ✅ Easy Compression: Reduce the size of your files effortlessly.
- ✅ Simple Decompression: Restore compressed files to their original state.
- ✅ User-Friendly: Designed with simplicity in mind, making it accessible for beginners.

---

## 🧰 Files in This Project

- `compressor.py` – Main script for compression & decompression.
- `zip_creator.py` – Helper script to create ZIP files.
- `README.md` – You're reading it!

---

## 🚀 Getting Started

### Prerequisites

Make sure you have **Python 3.x** installed. Download it from [python.org](https://www.python.org/downloads/) if needed.

### Installation Steps

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/rithish1233/File_Compression_Decompression_Project.git
   ```

2. **Navigate into the Project Folder:**

   ```bash
   cd File_Compression_Decompression_Project
   ```

3. **Run the Program:**

   ```bash
   python compressor.py
   ```

---

## 🖼️ Example Usage

### 🔹 Compressing a File

- Run: `python compressor.py`
- Select: **Compress**
- Enter the path of the file to compress
- The script will save a ZIP version of the file

### 🔹 Decompressing a File

- Run: `python compressor.py`
- Select: **Decompress**
- Enter the path of the ZIP file
- The script will extract and restore the original file

---

## 📌 Notes

- Double-check your file paths before running the script.
- Compressed files are saved in the same folder by default.
